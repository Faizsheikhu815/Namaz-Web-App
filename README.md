# Namaz-Web-App
Namaz Timing App

A comprehensive Islamic utility application that provides accurate prayer times and other useful Islamic features.

# Features

* 🕌 Accurate Namaz (Prayer) Timings
* 📍 Location-based prayer time calculation
* 🧭 Qibla Direction Finder
* 📅 Islamic (Hijri) Calendar
* 📖 Daily Quran Verses
* 🔔 Prayer Time Notifications
* 📿 Tasbeeh Counter
* 🌐 User-friendly Interface
# Technologies Used
* Frontend: HTML, CSS, JavaScript / React
* Backend: Node.js / Express
* APIs: Prayer Time API, Geolocation API
